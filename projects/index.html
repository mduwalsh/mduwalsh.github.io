Collective Action Problem: Oct, 2014 - May, 2015

Collective action problem model tries to solve problem with coordination efforts and payoffs in a group with hierarchial system through modeling and simulation. Simulation was agent based, and model included male and female group members, migration, group production, group conflicts, competition and extinction, and competition against nature. I developed simulation program in C, tested and debugged the program, wrote shell script to run multiple simulations in Linux cluster with Torque resource manager. Results were summarized for analysis by plotting graphs using gnuplot script embedded in C.
